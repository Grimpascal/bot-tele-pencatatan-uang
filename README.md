# Telegram Bot Project

Projek ini dibuat untuk membangun sebuah Telegram Bot.

## Rencana Teknologi
Berikut adalah pilihan teknologi populer yang bisa kita gunakan:
1. **Node.js (JavaScript/TypeScript)** dengan library **Telegraf** atau **GramJS**.
2. **Python** dengan library **python-telegram-bot** atau **aiogram** (async).

Silakan tentukan bahasa pemrograman dan library yang ingin Anda gunakan agar kita bisa melanjutkannya!
